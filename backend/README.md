## Taleb.tn - Backend API (Next Generation)

A modern rebuild of taleb.tn with new features and improved architecture. This is the foundation for our enhanced learning platform.

## Description

This is the backend API for the next generation taleb.tn platform, built with [NestJS](https://github.com/nestjs/nest) framework. Our focus is on creating a better learning experience with intelligent features that connect students.

### Key Features in Development

- **Study Partner Matching**: Intelligent algorithm to match students with compatible study partners
- Scalable and maintainable architecture
- per to per video calls integration to facilitate real-time study sessions
- Robust authentication and authorization
- Board to manage study groups and sessions
- real time Board for interactive learning

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
